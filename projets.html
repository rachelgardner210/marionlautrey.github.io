Projet lié au terrain de pétanque connecté

Projet réalisé en terminale STI2D : création d’un terrain de pétanque connecté. Objectif : permettre aux joueurs de jouer sans arbitre et d’éviter la triche.

  • Détection des points avec capteurs
  • Affichage des scores sur un écran
  • Mini application pour gérer les scores et vérifier les résultats

Technologies utilisées : Raspberry Pi / Python

Projet de site avec base de données

Projet réalisé durant ma deuxième année de BTS SIO (option SLAM) : création d’un site web connecté à une base de données.

  • Conception de la base de données sur phpMyAdmin et modélisation avec Sequelize
  • Développement d’API et tests unitaires avec Postman
  • Implémentation de logs pour le suivi des données
  • Développement du site web affichant les données de la base
  • Gestion des droits utilisateurs et administrateurs

Technologies utilisées : JavaScript, Node.js, Sequelize, Winston, SQL, HTML/CSS
